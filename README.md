# 🎮 Rock Paper Scissors Game | C++

A simple console-based game built using C++.  
This project was created to practice problem solving, clean code principles, and structured programming.

---

## 📌 Overview
The player selects the number of rounds (1–10), then competes against the computer by choosing:

- Rock
- Paper
- Scissors

The computer generates random choices each round, and at the end of the game a detailed summary is displayed showing:
- Player wins
- Computer wins
- Draws
- Final winner

---

## 🚀 Features
- Clean and organized code structure
- Random computer choices
- Round-by-round gameplay
- Final game statistics and winner summary
- Easy to read and maintain
